Difficultés et comment je les ai surmontées

  • Problèmes avec le SRC des images :

    • J'ai eu des problèmes avec le SRC des images lors de l'instantiation de ma classe. Je n'ai pas pu utiliser le meme path pour les Bestsellers de la page d'accueil et les Produits de la page des produits. J'ai compris que c'était parce qu'ils sont des fichiers séparés présents sur différents niveaux de dossiers. J'ai créé le dossier "Pages" pour que ce soit plus ordonné, alors j'ai décidé de le garder, mais pour résoudre mon problème, j'avais ajouté un deuxième path pour les images dans mon constructeur.

    J'ai continué a avoir de problèmes lors du déploiement sur Vercel. Cette problématique a été résolue quand j'ai changé le code pour utiliser les imports au lieu du path pour chaque image.

  • Ajout du paramètre isBestSellerProduct :

    • J'ai dû ajouter le paramètre isBestSellerProduct dans mon constructor pour pouvoir sélectionner seulement les quatre produits souhaités pour l'affichage sur la page d'accueil.

  • Maintien du footer en bas de page :

    • J'ai eu un peu de mal à maintenir le footer en bas de la page. J'ai trouvé une solution sur Stack Overflow :

    Sticky footer
  • Création de modales avec overlay en Tailwind :

    • J'ai eu des difficultés pour créer des modales avec overlay en utilisant Tailwind CSS. J'ai trouvé un article intéressant :

    Modal dialogue Tailwind
  • Problème avec la bannière sur mobile :

    • Sur la page d'accueil en mobile, la bannière apparaissait tout le temps sous le menu lors de son activation. J'ai dû utiliser -z-50 pour maintenir l'image à l'arrière.

  • Messages d'erreur/succès et validation des formulaires :

    • Pour afficher des messages d'erreur et de succès dans les formulaires, j'ai utilisé la méthode alert en JavaScript. Cette méthode crée une boîte de dialogue directement sur le navigateur demandant à l'utilisateur d'appuyer sur "OK" pour se fermer.

  • Validation des adresses e-mail en JavaScript :

    • J'ai rencontré des difficultés avec la validation des formulaires en JavaScript. Pour la validation des courriels spécifiquement, j'ai trouvé des posts intéressants sur Stack Overflow:

    Validation de formulaires
  • Source d'images et description des produits pour ce travail :

    • J'ai utilisé des images et des descriptions de produits de Chilli Beans :

    Chili Beans br
  • Configuration Vite pour deployment Vercel :

    • J'ai rencontré des difficultés avec la configuration du fichier vite.config.js, mes fichiers html ne sont pas dans le même dossier que l'index.html, cela fait que la configuration doit être différente. En regardant le vidéo fourni. par le professeur et après plusieurs tentatives et erreurs, j'ai réussi à trouver une configuration adéquate. Il faut utiliser le path complété en débutant par ../.
